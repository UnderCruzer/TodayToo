# 2026 피우다 글로벌 프로젝트
---

오늘도요 (今日もね) 

구조: React Native 앱 (어르신) + Next.js 웹 (가족·기관) + 공유 백엔드

팀원 : 이명진, 박성규, 박준혁

---

## 한 줄 요약
AI가 매일 먼저 말을 걸고, 어르신의 이야기가 쌓이면
한 달에 한 번 회고록 PDF를 가족에게 전달하는 시니어 케어 앱.
고독사·외로움·이상감지를 하나의 앱으로.

---

## 구현 순서

1. **모노레포 세팅** — Turborepo + 패키지 구조
2. **DB 스키마** — PostgreSQL + pgvector
3. **모듈 C 백엔드** — 회고록 생성 API (가장 핵심)
4. **웹 회고록 뷰어** — Next.js + window.print()
5. **모듈 A 백엔드** — 매일 안부 에이전트
6. **어르신 앱 홈 화면** — React Native 기본 UI
7. **모듈 B 백엔드** — 이상감지 + 가족 알림
8. **가족 대시보드** — Next.js 웹
9. **Cron 스케줄러** — 자동화 연결
10. **배포** — GCP Cloud Run (도쿄 리전)

---

## 해커톤 데모 시나리오

1. 어르신 앱 열기 → 오늘의 AI 메시지 확인
2. 답장 입력 → AI 후속 질문 실시간 확인
3. 옛날 사진 전송 → Vision 분석 후 연결 질문
4. 웹 브라우저에서 `/memoir/{id}` → 회고록 PDF 미리보기
5. "PDF로 저장" 버튼 → 실제 PDF 저장 시연
6. 가족 대시보드 → 어르신 상태 카드 + 알림 내역
